# Two-Player Snake Game (Python/Tkinter)

**Description:**  
A two-player Snake game built with Python and Tkinter. Features:
- Single-player and multiplayer mode
- Lives system and score tracking
- Pausing and resuming
- Dynamic speed increase as food is eaten
- Border collision detection
- Game over screen with winner display

**Controls:**  
- Player 1: Arrow Keys  
- Player 2: WASD  
- Pause: P

**Technologies:** Python, Tkinter
